Claude Code / tmux

Claude Code 做到一半斷線？tmux 係救命工具

呢張圖重點講得好實際：如果你長期喺遠端主機、Linux、macOS 上跑 Claude Code，tmux 幾乎係必備。原因唔只係方便，而係可以避免中途斷線、誤關 terminal、或者多個 session 同步工作時整個 workflow 崩掉。

重點摘要

  • 作者指出，很多人會在遠端機器上跑 Claude Code，但主控端可能係 Windows/macOS，一開太多 terminal 視窗就容易亂。
  • 最慘情況係 Claude Code 跑到一半，手滑關咗 terminal，之前燒咗嘅 token 同工作進度就一齊蒸發。
  • 因此作者長期保持一個習慣：一定在 tmux 裏面啟動 Claude Code。
  • tmux 對配合 Claude Code 嘅 Swarms / Subagents 特別有用，因為即使主控端視窗關咗，背景 session 仍可繼續跑。
  • 做法上可以先開 tmux session，再分 pane，將 Claude Code 放入其中一格，由它持續執行後續任務。

安裝方式

macOS： brew install tmux
Ubuntu： sudo apt install tmux

一句帶走

如果你會長時間跑 Claude Code，尤其係遠端開發、子代理、多 session 並行，tmux 唔係 optional，係保命層級工具。

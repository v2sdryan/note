Agent 專案本週觀察

呢張圖整理咗近期幾個熱門 Agent 專案，重點唔係單一模型升級，而係「可協作、可執行、帶完整方法論」嘅系統級 Agent 正快速冒起。作者認為，真正能改變產業爆發點嘅，係有完整工程流程、記憶、技能同角色分工嘅 Agent 基建。

重點摘要

  • Superpowers 被列為本週增長最強項目，主打把完整軟體開發流程（brainstorm → spec → planning → TDD → review）包裝成真正工程師級工作流。
  • everything-claude-code 被視為 Claude Code 專屬性能優化方案，重點在 skills、instincts、memory、安全機制同 research-first 開發體系。
  • Agency Agents 強調用一句話就能搭出完整 AI 公司，透過多個專業角色代理人組成可交付團隊。
  • MiroFish 著重群體智慧預測，想用大量可互動 Agent 去做金融、市場、社會事件等預測。
  • Lightpanda Browser 是面向 AI Agent 的無頭瀏覽器，目標係更快、更省記憶體，解決網頁操作卡頓問題。

一句帶走

作者嘅判斷係：2026 真正爆發嘅，會係 Agent 基礎設施同公司級玩法，而唔單止係模型分數再高少少。
